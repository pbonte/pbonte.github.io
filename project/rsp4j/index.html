RSP4J: An API for RSP

Apr 20, 2021
PDF Repo

RSP4J is a library to build RDF Stream Processing (RSP) Engines according with the reference model RSP-QL.

RSP4J is inspired by the OWL API, and other works that aim at spreading the Semantic Web (Stream Reasoning) research by means of practical and usable software tools.
